# Asset Library Documentation

## 1. Logo Suite
- **Source:** Custom created for Content Flow Studio.
- **License:** Proprietary / Project-exclusive.
- **Variations:**
  - `logo-full.svg`: Primary horizontal logo for light backgrounds.
  - `logo-white.svg`: Primary horizontal logo for dark backgrounds.
  - `logo-icon.svg`: Monogram for avatars and mobile navigation.
  - `logo-wordmark.svg`: Text-only version for footer and minimal UI.
  - `favicon.svg`: High-contrast icon for browser tabs.

## 2. Platform Icons
- **Source:** Custom minimalist set.
- **License:** CC0 (Public Domain Equivalent).
- **Usage:** Monochromatic icons for dashboard integrations (LinkedIn, X, YouTube, Instagram, etc.).

## 3. UI Icons
- **Source:** Custom minimalist set.
- **License:** CC0.
- **Usage:** Functional arrows, checks, and utility symbols.

## 4. Illustrations
- **Source:** Code-first SVG illustrations.
- **License:** MIT.
- **Usage:** Technical dashboard and pipeline visualizations for landing pages and onboarding.

## 5. Patterns
- **Source:** CSS-driven geometric patterns.
- **License:** MIT.
- **Utilities:**
  - `.bg-mesh`: Blurred premium gradient.
  - `.bg-dots`: Enterprise dot matrix.
  - `.bg-grid`: Technical grid background.

---
**Guidelines:**
- Do not alter SVG aspect ratios.
- Maintain a minimum clear space around the logo equal to the width of the central icon node.
- Prefer monochrome platform icons over branded color versions for a unified enterprise feel.
